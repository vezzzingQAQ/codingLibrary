弱类型的一些问题

  • 运行阶段才能发现异常
    ex:obj.foo();//并没有foo方法
  • 函数传参类型不确定

强类型优势

  • 错误更早暴露
  • 代码更智能,编码更准确
    ex:根据对象类型提示拓展
  • 重构更加牢靠
